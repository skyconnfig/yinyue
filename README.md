# 音乐 + 倒计时

活动策划用背景音乐和倒计时软件集合。

## 目录结构

- `yinyue/` - 分类背景音乐
  - 1、暖场/ - 暖场音乐
  - 2、倒计时/ - 倒计时背景音乐
  - 3、上场/ - 登场音乐
  - 4、领导-嘉宾/ - 领导和嘉宾上场音乐
  - 5、签约/ - 签约仪式音乐
  - 6、抽奖/ - 抽奖环节音乐
  - 7、仪式/ - 颁奖、揭牌等仪式音乐
- `音乐/` - 备用音乐集合
- `X月X日音频文件/` - 备用音频文件
- `倒计时/` - FlyClock 倒计时软件
- `sportssoundspro/` - Sports Sounds Pro 音效软件

## 使用说明

各目录音乐按活动流程分类，可根据具体场景选择对应音乐。
